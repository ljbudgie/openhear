## ITE Shell Bill of Materials (Per Ear)

Approximate costs for a single ITE shell build. Double for binaural.

| Item | Quantity | Approx. Cost | Notes |
|------|----------|-------------|-------|
| Biocompatible resin (per shell) | ~5–10mL | £1–£3 | Cost depends on resin brand |
| Hydrophobic coating (per application) | ~1mL | £1–£2 | From a larger bottle — lasts many shells |
| Silicone impression kit | 1 kit (enough for 2–4 impressions) | £15 | One-time purchase for both ears |
| Sandpaper set (400–2000 grit) | 1 set | £5 | Reusable across multiple shells |
| IPA 99% (per wash cycle) | ~50mL | £0.50 | Buy in 5L containers for economy |
| Pigment paste | ~0.1mL per shell | £0.50 | From a larger container |
| Matte clear coat | Minimal per shell | £0.50 | From a spray can — lasts dozens of shells |
| **Wax guard filters** (Phonak CeruShield or equivalent) | 1 | £0.50 | Consumable — replace monthly |
| **Total per shell (materials only)** | | **£5–£15** | |
| **Total per shell (with impression kit, first time)** | | **£20–£30** | |

> **Cost does not include the printer** (£200–£300 one-time) or electronics (see [BOM](../BOM.md)). After the initial investment, each replacement shell costs £5–£15 in materials. Compare to £200–£500 for a commercial replacement shell from a hearing aid lab.

## Future Roadmap Ideas

These are ideas for the community to explore. None are implemented yet.

### Near-Term (2025–2026)

| Idea | Description | Difficulty |
|------|-------------|-----------|
| **Community scan library** | Anonymised ear canal scan repository — learn from diverse ear anatomy, create statistical shell templates | Medium |
| **Auto-shelling plugin** | Blender/FreeCAD plugin that automatically converts a raw ear scan into a shelled, channelled ITE body | High |
| **Multi-material ITE shells** | Multi-resin printing (rigid body + flexible canal tip in one print) using dual-vat printers | Medium |
| **Standard faceplate system** | Universal faceplate dimensions so electronics modules can snap between different shell bodies | Low |
| **Colour-matching app** | Phone app that photographs your skin tone and calculates resin pigment ratios | Medium |

### Medium-Term (2026–2027)

| Idea | Description | Difficulty |
|------|-------------|-----------|
| **Temperature-adaptive canal tips** | Shape-memory polymer canal tips that soften at body temperature for self-adjusting fit | High |
| **Integrated antimicrobial surfaces** | Silver or copper nanoparticle resins with built-in antimicrobial properties, verified by testing | Medium |
| **Wireless charging integration** | Qi coil embedded in the faceplate for wireless charging without removing the shell | High |
| **Biodegradable shell option** | PLA-based biocompatible resin for environmentally conscious shell production | Medium |
| **AI-assisted shell design** | ML model trained on successful shell designs that suggests modifications based on ear scan geometry | Very High |

### Long-Term (2027+)

| Idea | Description | Difficulty |
|------|-------------|-----------|
| **Injection-moulded silicone shells** | Desktop silicone injection moulding for fully soft ITE shells — ultimate comfort | High |
| **Bioprinted canal liners** | Hydrogel-based canal liners printed from biocompatible hydrogels — closest possible to natural tissue | Very High |
| **Open scan-to-shell pipeline** | Complete open-source pipeline: scan ear → auto-generate shell → auto-slice → print — zero manual CAD | Very High |
| **Shared fitting database** | Open database of successful shell parameters indexed by ear scan morphology — new users start from a good baseline | High |

## How to Contribute

This is an open-source project. Every contribution makes the next person's shell better.

### Ways to Help

- **Print a shell and share your experience** — open an issue with your printer, resin, settings, and results
- **Share your ear scan workflow** — which app/scanner worked best for you?
- **Test a coating** — try a new hydrophobic coating and report performance
- **Design a better faceplate** — snap-fit, magnetic, threaded — share your CAD files
- **Write a tutorial** — video or written, any language
- **Review the safety documentation** — if you are a clinician or audiologist, your review is invaluable
- **Translate** — help make this guide accessible in other languages

### Contribution Guidelines

1. Fork the repository
2. Create a branch for your contribution
3. Submit a pull request with a clear description of what you changed and why
4. Include photos/data where relevant — a photo of a printed shell is worth a thousand words
5. Follow the existing documentation style — professional, accessible, honest
