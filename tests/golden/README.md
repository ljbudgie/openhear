# Golden DSP vectors

These files are the **frozen reference** for OpenHear's DSP stages.
They are generated by `python -m scripts.benchmark --regen-golden`
and consumed by `tests/test_golden_vectors.py` and the benchmark
harness's parity check.

If a code change in `dsp/` causes any of these vectors to diverge,
that is **by definition** an algorithmic change and must be
accompanied by an explicit regeneration commit and a CHANGELOG
entry.  Do not regenerate silently.

Files:

* `input.npy` — deterministic float32 input frames (seed 0xdeadbeef).
* `noise_reduction.npy` — `SpectralSubtractor` output.
* `compression.npy` — `WDRCompressor` output.
* `voice_clarity.npy` — `VoiceClarityEnhancer` output.
* `feedback.npy` — `FeedbackCanceller` output.
